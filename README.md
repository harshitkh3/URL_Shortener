<h1>URL Shortener in NodeJS</h1>
<h4>
  I have made a URL Shortener using nano id module in NodeJS with MongoDB as a Database.
</h4>
<h5>
  Routes
</h5>
<p>
  POST -  http://localhost:8000/url  - create a new shortID by sending a url in the following json format in the body</br>
{\
    "url":"https://www.wikipedia.org/"\
}
</p>
